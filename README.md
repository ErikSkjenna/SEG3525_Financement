# BureauZen — Devoir 4 Commerce électronique

Prototype haute-fidélité pour SEG3525, construit avec :

- `index.html`
- `css/styles.css`
- `js/app.js`
- Vue 3 par CDN, donc aucun fichier `.jsx`

## Comment ouvrir le site

1. Ouvrir `index.html` dans un navigateur moderne.
2. Garder une connexion Internet active, car Vue est chargé par CDN.

## Exigences couvertes

- Cadre JavaScript : Vue 3
- Recherche à facettes fonctionnelle
- Catalogue de produits
- Panier
- Achat guidé par étapes : panier, informations, paiement fictif, confirmation
- Sondage utilisateur
- Communication verbale : promotion, information, connexion
- Conception visuelle cohérente : couleurs, hiérarchie, cartes, espace négatif, contraste

## Idée du commerce

Nom : BureauZen
Type : boutique e-commerce d’accessoires ergonomiques et d’organisation pour étudiants et travailleurs.
